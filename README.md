# shopping-cart

A React project that designs and implements a mock e-commerce shopping cart frontend, demonstrating the following tools and concepts:

- React Hooks (`useState` and `useEffect`)
- Test-Driven Development (TDD) with Vitest and React Testing Library (RTL)
- Mocking in tests
- PropTypes for runtime type checking
- React Router (routing, nested routes, and outlet context)
- Data fetching
- CSS Modules

Users can browse products, filter by product category, adjust quantities, and manage items in a shopping cart, with product data fetched from an external API.
